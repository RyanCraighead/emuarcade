# Security Policy

## Supported Versions

Security fixes target the `main` branch until formal releases are created.

## Reporting a Vulnerability

Please report security issues privately to the project maintainers. If the repository has GitHub private vulnerability reporting enabled, use that first. Otherwise, open a minimal public issue that asks for a maintainer contact without including exploit details.

## Data Boundary

EmuArcade is designed so ROM and BIOS files remain in the user's browser. Reports about accidental file upload, server-side file retention, unsafe URL handling, or cross-origin behavior are high priority.
